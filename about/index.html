About Me

Hi! I'm Benjamín Muľár, an university student.

I'm in my first year of university & programming is my passion. The first thing I learned (early to mid 2018) while studying IT was creating websites - HTML and then shortly after CSS.


I began self-learning Python during the summer months of 2018 & later on in 2019 moving to C++ basics, but then ended up coming back to Python in school (2020).


In 2021, I proceeded to learn core Java & shortly after it Haxe. That's when I began my gamedev journey. I've created my first game "Égaré" and Ludum Dare game "Snowcastle Meltdown" during 72h.


After a "small" pause, I started The Odin Project web dev course in 2022, getting back to web development & improving myself, especially finally learning JavaScript & finishing some JS projects.


I'm currently continuing learning web development (The Odin Project) & game development.
